# Выделение окошка

Chrome-расширение для быстрого скриншота выделенной области — прямо в браузере.

Создано для удобной работы с ИИ-чатами (Claude, ChatGPT, DeepSeek и др.):  
выделил нужную часть экрана → вставил в чат через Ctrl+V.

![Демо](https://raw.githubusercontent.com/YOUR_USERNAME/screenshot-extension/main/demo.gif)

---

## Как это работает

1. Нажми иконку расширения в панели браузера
2. Нажми **«Выделить область»**
3. Зажми мышь и потяни прямоугольник вокруг нужной части экрана
4. Отпусти — скриншот скопируется в буфер обмена
5. Вставь в чат через **Ctrl+V**

> Размер выделенной области отображается рядом с курсором в реальном времени.  
> Отмена — **Esc**.

---

## Установка

> Расширение не опубликовано в Chrome Web Store, поэтому устанавливается вручную.

1. Скачай репозиторий: **Code → Download ZIP** → распакуй
2. Открой Chrome и перейди на страницу `chrome://extensions`
3. Включи **«Режим разработчика»** (переключатель в правом верхнем углу)
4. Нажми **«Загрузить распакованное»** и выбери папку с расширением
5. Готово — иконка появится в панели браузера

---

## Ограничения

Расширение **не работает** на системных страницах Chrome:
- `chrome://extensions`, `chrome://settings` и другие `chrome://` страницы
- Chrome Web Store
- PDF-файлы, открытые в браузере

Это ограничение безопасности самого Chrome, не баг расширения.  
На всех обычных сайтах работает корректно.

## Зачем, если есть Win+Shift+S?

| | Win+Shift+S | Это расширение |
|---|---|---|
| Где работает | Весь экран, любые окна | Только в браузере |
| Системные страницы Chrome | ✅ | ❌ |
| Шагов до вставки в чат | 4–5 | 3 |
| Установка | Встроено в Windows | Этот README |

Когда ты уже в браузере и в чате — расширение быстрее.

## Технологии

- Manifest V3
- `chrome.tabs.captureVisibleTab` для захвата экрана
- `navigator.clipboard.write` для записи в буфер
- Чистый JS, никаких зависимостей
